# Psychic 5 for MiSTer
Psychic 5 compatible core for MiSTer FPGA

## Features
* Full emulation of the original PCB
* NTSC-friendly mode
* adjustable H/V sync


## Inputs
* Supports SNES pad type currently
* SNAC(WIP)
